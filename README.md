## Opis

Pomocnik przy tworzeniu stron parafialnych opartych na wordpresie. Za jego pomocą szybko na stronie udostępnisz takie informacje jak:
 * Intencje mszalne
 * Ogłoszenia parafialne
 * Rozpiska odzwiedzin kolędowych
